# go-mydns

CLI for mydns.jp

## Usage

```
Usage of mydns:
  -at value
        add new TXT entry (-at foo=bar)
  -ut value
        update TXT entry (-ut foo=bar)
```

## Installation

```
$ go get github.com/mattn/go-mydns/cmd/mydns
```
